# SyncWatch Workspace

Repository layout:

- `mobile/`: Expo React Native client app
- `backend/`: Firebase backend workspace, docs, and extension scaffold

Quick start:

1. Mobile app:
   - `cd SyncWatch/mobile`
   - `npm install`
   - `npm run start`
2. Backend functions:
   - `cd /SyncWatch/backend/functions`
   - `npm install`
   - `npm run build`
