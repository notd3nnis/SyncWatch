# SyncWatch

## Repository Structure

```
SyncWatch/
├── mobile/       # Expo React Native client app
└── backend/      # Firebase backend workspace, docs, and extension scaffold
```

## Quick Start

### Mobile App

```bash
cd SyncWatch/mobile
npm install
npm run start
```

### Backend Functions

```bash
cd SyncWatch/backend/functions
npm install
npm run build
```
